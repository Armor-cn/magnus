<p align="center">magnus</p>

`dota` 半人猛犸，干掉`prisma`中的反人类设计


## 功能

- 使用 Graphql 请求数据库
- 根据装饰器自动生成 graphql、grpc（proto）
- 使用 protobuf 加速序列化
- 支持 typeorm 及其他适配器
- 自动同步数据库字段（typeorm）
